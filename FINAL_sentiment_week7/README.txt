1. run classifier.py - it fit classifier model with train data - final.csv that consist of approximately
   750 negative and 750 positive reviews on mobile phones. Than write classifier object into file 
   dumped_lin_SVC_classifier.pkl (already done for python2 but not for python3)
2. install flask, flask-bootstrap and flask-wtf libraries
3. templates folder with html pages; main page - index.html
4. run hello2.py, write in browser http://127.0.0.1:8080/, enter review in russian or ukrainian 
   languages with phone brands in latin. 
   As an example - several texts from test dataset of week_6
